# profile-helper
![GitHub last commit](https://img.shields.io/github/last-commit/the8ff8/profile-helper) ![License](https://img.shields.io/badge/license-MIT-blue)

A powerful CLI tool designed to automate GitHub workflows, boost developer profiles, and seamlessly unlock GitHub achievements via automated commits and PRs. Built by [@the8ff8](https://github.com/the8ff8).

## Usage
```bash
# 1. Make scripts executable
./scripts/setup.sh

# 2. Run the master unlocker script
./scripts/unlock-all.sh

# 3. Or run the interactive Node.js CLI
npm start
```
